## Instructions
1. Clone the repository first and install the requirements packages :
   
   ```bash
   git clone https://github.com/mahdiadham/go-corona.git
   cd go-corona
   npm install
   ```
2. Start editing project with command :
   
   ```bash
   npm run build
   ```
3. Run project with live server and enjoy it !
